This project is a simple Contacts Management System developed using Python and MySQL Connector. It allows users to efficiently manage contact information by performing basic database operations.

The application supports:

Adding new contacts

Viewing saved contacts

Updating existing contact details such as phone number, email, and home address

Deleting contacts when no longer needed

Python is used for handling user interaction and program logic, while MySQL is used for securely storing and managing contact data. This project demonstrates the use of CRUD operations (Create, Read, Update, Delete)
